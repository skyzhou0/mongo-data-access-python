# mongo-data-access-python

## Setup & Installation
1. Create virtual environment
```Bash
python3 -m venv mongo-venv 
source mongo-venv/bin/activate
```
2. Install pymongo
```Bash
pip install -r requirements.txt
or 
pip install 'pymongo[srv]' 
```
